## Usage Notes

### UTM Parameter Legend
- **utm_source** - Where the traffic came from (platform/channel)
- **utm_medium** - Type of marketing medium (social, email, referral)
- **utm_campaign** - Campaign name (waitlist-launch)
- **utm_content** - Specific content variant (optional, for detailed tracking)

### Influencer Template
For each influencer partnership, replace `{{influencer_name}}` with their actual name or handle:
- Example: `https://psalmix-waitlist.vercel.app?utm_source=sarah_jones&utm_medium=referral&utm_campaign=waitlist-launch`
- Example: `https://psalmix-waitlist.vercel.app?utm_source=thegospelbeat&utm_medium=referral&utm_campaign=waitlist-launch`

### Implementation Tips
1. **Shorten URLs** - Use bit.ly or similar to create shortened versions for social media and SMS
2. **QR Codes** - Generate QR codes from these UTM links for print/visual media
3. **Analytics** - Monitor utm_source and utm_content in Google Analytics to track channel performance
4. **A/B Testing** - Create variations with different utm_content values to test message effectiveness
