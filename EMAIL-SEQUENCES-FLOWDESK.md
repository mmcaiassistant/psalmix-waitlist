## FLOWDESK IMPLEMENTATION NOTES

### Placeholders Used (Copy/Paste Ready)
```
{{first_name}} - Subscriber's first name
{{referral_link}} - Unique referral link
{{position}} - Current waitlist position
{{referral_count}} - Number of referrals completed
```

### Automation Triggers in FlowDesk

**Welcome Sequence:**
- Email 1: Trigger on subscription join (immediate)
- Email 2: Delay 2 days, then send
- Email 3: Delay 4 days, then send

**Milestone Emails:**
- 1st Referral: Trigger when `referral_count = 1`
- 3 Referrals (Beta): Trigger when `referral_count = 3`
- 10 Referrals (VIP): Trigger when `referral_count = 10`
- 25 Referrals (Golden Ticket): Trigger when `referral_count = 25`

**Pre-Launch Email:**
- Trigger: 48 hours before launch date (manual schedule)
- Segment: All active waitlist members
- Send time: Same time as Day 1 welcome email (for consistency)

- [ ] Test with various name lengths (short/long)
- [ ] Verify "Reply to" email address is set
- [ ] Check unsubscribe link visibility
