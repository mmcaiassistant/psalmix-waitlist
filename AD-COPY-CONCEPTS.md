## 📊 Testing Framework

### Phase 1: Test Hooks (Week 1)
Run 3-4 ads with different hooks, same audience.
Budget: $25-50 per ad.
Goal: Identify best-performing hook.

### Phase 2: Test Audiences (Week 2)
Take winning hook, test across different audiences.
Budget: $50-100 per audience.
Goal: Identify best-converting audience.

### Phase 3: Scale Winners (Week 3+)
Scale winning hook + audience combinations.
Budget: $50-100/day on winners.
Goal: $1-3 cost per waitlist signup.
